# Follow_Me
## Deep-Learning Project
In this Project, I built and trained a Fully-Convolutional Neural Network to find a specific person in images from a simulated quad-copter.

## Requirements 
  * Python 3.4
  * Tensorflow 1.2.1
  * python-socketio
  * scikit-image
